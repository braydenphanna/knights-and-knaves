# Knights and Knaves
## Gameplay
- Turn Based Combat System
- World Map
- Each npc is either a knight or knave
  - Knights: can only tell the truth
  - Knaves: can only lie
- Recruitable party members who are knights/knaves
- Minigames/Puzzles
