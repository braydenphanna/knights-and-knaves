# Knights and Knaves
 
## Tech Stack
- Godot 
- C#
- Blender 

## Gameplay
- Turn based combat
- Each npc is either a knight or knave
  - Knights: can only tell the truth
  - Knaves: can only lie
- Recruitable party members who are knights/knaves
- Player character either
  - Can choose knight or knave
  - neither (leaning towards this)
- Logic puzzles
- Minigames
